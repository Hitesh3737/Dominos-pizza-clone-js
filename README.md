## âœ¨ Key Features

*   **Responsive Design:** Fully fluid layout that adapts from Desktop to Mobile (iPhone 16 view) using CSS Flexbox and Grid.
*   **Dynamic Menu System:** Menu items are rendered dynamically using JavaScript objects.
*   **Functional Shopping Cart:**
    *   Add items to the cart.
    *   Increase/Decrease quantity.
    *   Automatic total calculation.
    *   Cart state persists using LocalStorage.
*   **Coupon Logic:**
    *   Apply code `DOM50` for a 50% discount.
    *   Apply code `PARTY3` for a free Garlic Bread (when buying 3+ pizzas).
*   **Interactive UI:**
    *   Custom 3D animated Cart button.
    *   Mobile Hamburger navigation.
    *   Hover effects and transitions.
*   **Location Feature:** Simulates a "Drive-N-Pick" feature that redirects to Google Maps based on user location.
